# Mental Health Support Web App

This is a full-stack web application built as a group project to provide basic mental health support and user interaction features.

The goal was to create a platform where users can sign up, log in, share their thoughts, and connect with support services in a simple and accessible way.

## What it does

- User authentication (signup & login)
- Users can share their thoughts/comments
- Profile information can be stored
- Users can submit queries for support
- Basic package/consultation handling system
- Backend stores and manages all data using a database

## Tech Used

- Frontend: React (JavaScript, CSS)
- Backend: Node.js with Express
- Database: MySQL

## Project Structure

- `frontend/` → User interface built with React
- `backend/` → Server, APIs, and database handling

## How to run

### Backend
cd backend  
npm install  
node server.js  

### Frontend
cd frontend  
npm install  
npm start
